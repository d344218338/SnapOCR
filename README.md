# SnapOCR - 免费截图OCR工具

截图即识别，完全免费，无需登录，无需API密钥，本地离线可用。

## 功能

| 功能 | 说明 |
|------|------|
| 截图识别 | 按 F4 框选屏幕任意区域，自动识别文字 |
| 表格识别 | 识别表格结构，输出整齐的 Markdown 格式 |
| 翻译 | 识别后一键翻译（需本地 Ollama，可选） |
| 高亮标注 | 在截图上画框，框内正常、框外变暗，突出重点 |
| 从文件识别 | 打开图片文件直接 OCR |

## 一键安装（Windows）

1. 下载本项目，解压
2. 双击 `SnapOCR-一键安装.bat`
3. 等待自动安装完成
4. 双击桌面「SnapOCR 截图识别」即可使用

## 使用方法

1. 启动后按 **F4** 进入截图模式
2. 拖拽鼠标框选要识别的区域
3. 松开鼠标，自动识别文字
4. 可以复制文字、翻译、或点击「高亮标注」标注重点

## 高亮标注

截图后点击「高亮标注」，在图片上拖拽画框：
- 框内区域正常显示
- 框外区域自动变暗
- 可以画多个框，支持撤销
- 结果可复制到剪贴板或保存为图片

## 技术栈

- OCR引擎：RapidOCR (PaddleOCR ONNX)，本地运行，无需网络
- 界面：PyQt6 深色主题
- 翻译：Ollama 本地大模型（可选功能）

## 系统要求

- Windows 10/11
- 约 500MB 磁盘空间（含 OCR 模型）
